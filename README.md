# ALE_analysis
Run script in directory containing the transfer frequency files (ending .uTs) from the ALE gene tree-species tree reconciliations.

## Usage
```bash
sh calc_from_to_T.sh
```
## Output file: From_to_T.txt
Example output
```
Branch	From	To
100	46.56	81.97
101	81.8	84.7
102	181.39	196.41
103	178.59	206.8
```
